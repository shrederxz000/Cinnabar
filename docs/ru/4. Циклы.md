Цикл объявляестся ключевым словом loop

```
void main() {
    loop {
        ... // тело цикла
    }
}
```

Бесконечный цикл:

```
void main() {
    loop {
        continue // если не писать, то будет ошибка (сделано от непренднамереных бесконечных циклов)
    }
}
```

plug std.io
