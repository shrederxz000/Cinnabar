Все типы данных языка

Целочисленные:

```
int // (uint = uint32, uint = uint64 по необходимости)
int8 
int16
int32
int64
int128
```
Целочисленные неотрецательные:

```
uint  // (uint = uint32, uint = uint64 по необходимости)
uint8
uint16
uint32
uint64
```
С плавающей точкой:

```
float32
float64
```

Логические:
```
true
false
```

Строковые:

```
string  // записываются в ""
```

Символьные:
```
char // записываются в ''
```
